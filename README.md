# 履歷 (中文/English)

## 專案資訊
- 專案名稱：profile
- 版本：v1.0.0
- 類型：模組 (ES 模組)
- 開發環境：Node.js (建議使用版本 14 或以上)

## 開發
```
npm install
```

## 資料改寫
- 畫面資料由 src / lang 的中英文資訊產生
  - 目前只支援 en.json(英文) / zh.json(中文)

## 建置
```
npm run build
```
- dist 資料夾就是完成的專案

## 技術堆疊

- Vue 3
- TypeScript
- Vite
- Eslint
- TailwindCSS
- Element Plus
